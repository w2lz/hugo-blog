## 描述

这个题目说的是，给你一个非负整数 n，你要写一个函数返回第 n 个斐波那契数。其中斐波那契数列最开始的两项是 0 和 1，后面任意一项都是它前面两个数字之和。

```markdown
比如说，你写的函数是 f，那么就有：

f(0) = 0,
f(1) = 1,
f(2) = 0 + 1 = 1,
f(3) = 1 + 1 = 2,
f(4) = 1 + 2 = 3,
f(5) = 2 + 3 = 5,
...

以此类推。
```
