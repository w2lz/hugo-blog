## 描述

这个题目说的是，给你一个正整数，你要将它转换成 Excel 表格中的列标题。

```markdown
// Excel 表格中，列编号与列标题的对应关系
  1 -> A
  2 -> B
  ...
  26 -> Z
  27 -> AA
  28 -> AB
  ...

比如说，给你的正整数是 1，你要返回的列标题是字符串 A：

1 -> A

再比如说，给你的正整数是 28，你要返回的列标题是字符串 AB：

28 -> AB
```
