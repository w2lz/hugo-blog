## 描述

这个题目说的是，你要设计一个支持 add 操作和 find 操作的类。其中，add 操作添加一个整数到这个类的内部数据结构；而 find 操作则是检查内部数据结构中是否存在求和等于给定值的两个数。

```markdown
比如说，一开始添加 3 个数字：

add(1)
add(2)
add(3)

你可以用 find 检查内部数据结构中是否存在两个数字相加等于 4 或 6：

find(4) => true
find(6) => false
```
