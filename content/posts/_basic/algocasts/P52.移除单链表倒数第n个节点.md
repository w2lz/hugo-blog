## 描述

这个题目说的是，给你一个单链表和数字 n，你要移除单链表倒数第 n 个节点，然后返回单链表。

```markdown
比如说，给你的单链表是：

1 -> 2 -> 4 -> 8

给你的数字 n 是 3，单链表中倒数第 3 个节点是 2，移除 2 以后的单链表是：

1 -> 4 -> 8
```
