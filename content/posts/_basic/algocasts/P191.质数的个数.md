## 描述

这个题目说的是，给你一个非负整数 n，你要计算出小于 n 的质数（素数）一共有多少个。

```markdown
比如说，给你的 n 等于 8：

n = 8

小于 8 的质数一共有 4 个，分别是：

2, 3, 5, 7

因此你要返回 4。
```
