## 描述

这个题目说的是，给你一个非负整数数组，数组中的整数表示直方图的高度，每个直方图的宽度都为 1，你要计算出直方图中最大矩形的面积。

```markdown
比如说，给你的数组是 2, 1, 3, 4, 1, 它对应的直方图如图所示：

          +--+
          |  |
       +--+  |
       |  |  |
 +--+  |  |  |
 |  |  |  |  |
 |  +--+  |  +--+
 |  |  |  |  |  |
 +--+--+--+--+--+-->
  2  1  3  4  1

在这些直方图组成的区域里，可以形成的最大矩形是 3 和 4 这两个直方图中，高度为 3，宽度为 2 的矩形。它的面积为 6，因此你要返回 6。
```
